# 1. 정역학 (Statics) — 가장 중요

## 핵심 개념

### 1.1 평형 (Equilibrium)

- 조건:
    

```id="eq1"
ΣF = 0
ΣM = 0
```

👉 매우 중요:

- 힘의 합 = 0
    
- 모멘트 합 = 0
    

---

### 1.2 벡터 vs 스칼라

|구분|특징|
|---|---|
|벡터|크기 + 방향|
|스칼라|크기만|

- 힘, 속도 → 벡터
    
- 에너지, 온도 → 스칼라
    

👉 MCQ 자주 나옴:

- “다음 중 벡터는?”
    

---

### 1.3 벡터 연산

**내적 (Dot product)**

```id="eq2"
A · B = |A||B|cosθ
```

- 결과: 스칼라
    

👉 출제:

- 각도 포함 문제
    
- 직교 → 0
    

---

## 출제 포인트

- 평형 조건 (100% 출제)
    
- 벡터 vs 스칼라 구분
    
- 내적 의미
