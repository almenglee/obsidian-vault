# 🧠 2) 시험 대비 요약본 (핵심 압축 + 암기용)

## 📌 상관관계 핵심

- -1 ~ 1 범위
- 피어슨: 선형 관계
- 켄달: 순위 기반
- 핵심: “상관 ≠ 인과”

## 📌 회귀분석 핵심

- Y = β₀ + β₁X + ε
- β₁ = X 1 증가 시 Y 변화량
- R² = 설명력 (0~1)

## 📌 다중회귀 핵심

- 변수 여러 개 사용
- β는 “다른 변수 통제된 효과”
- 다중공선성 = 변수 간 높은 상관 → 문제 발생

## 📌 통계 검정 핵심

- t-test: 개별 변수
- F-test: 전체 모델
- p < 0.05 → 유의

## 📌 반드시 구분

- 상관분석: 관계 “있나?”
- 회귀분석: “얼마나 영향?”

## 📌 시험 단골 포인트

- 상관관계 ≠ 인과관계 (최중요)
- R² 해석
- p-value 의미
- 다중공선성(VIF)
- 회귀계수 해석
