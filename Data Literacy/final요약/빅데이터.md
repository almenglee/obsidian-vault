# 🧠 2) 시험 대비 요약본 (핵심 압축 + 암기용)

## 📌 빅데이터 정의

- 기존 DB로 처리 불가
- TB~PB 이상
- 3V = Volume / Velocity / Variety

## 📌 3V 핵심

- Volume: 크기
- Velocity: 속도
- Variety: 다양성 (비정형 포함)

## 📌 데이터 종류

- 정형: DB/엑셀
- 반정형: XML/HTML
- 비정형: 이미지/영상/SNS (80%)

## 📌 핵심 특징

- 비정형 중심
- 분산처리 필요
- AI 분석 기반
- 21세기 원유

## 📌 데이터 자본 특징 (시험 단골)

- 비경쟁성
- 비대체성
- 재사용 가능
- 가치 누적 증가

## 📌 빅데이터 가치

- 비용 절감
- 의사결정 개선
- 예측 정확도 증가
- 공공/기업 모두 효과

## 📌 분석 기술

- 텍스트 마이닝
- 감성 분석 (opinion mining)
- 군집 분석
- SNA (네트워크 분석)

## 📌 도구

- Hadoop = 분산 처리
- R = 통계 + 시각화

## 📌 핵심 사례

- Netflix = 추천 시스템
- ZARA = 데이터 기반 생산
- Google Trends = 검색 분석

## 📌 시험 핵심 포인트

- 3V 정의
- 비정형 데이터 비율 (80%)
- 데이터 자본 특징 4가지
- Hadoop 역할
