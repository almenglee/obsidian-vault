# 🧠 시험 대비 요약본 (초압축)

- 설문조사 = 의견 → 데이터 수집 도구
- 목적: 통계 + 의사결정
- 방법: 면접 / 전화 / 집단 / 우편 / 전자
- 핵심 구조: 연구 → 가설 → 표본 → 설문 → 분석
- 질문 유형:
    - 개방형 = 심층
    - 폐쇄형 = 분석용
    - 부수질문 = 조건부
- 설문 핵심:
    - 표준어
    - 중립성
    - 간결성
    - 1문항 1개념
- 금지:
    - 유도 질문
    - 모호한 표현
    - 전문용어 남용
- 순서:
    - 쉬운 → 어려운
    - 일반 → 민감
- 핵심 원칙: 설문 품질 = 분석 결과 품질
