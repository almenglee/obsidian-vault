# 📚 교본용 정리본 (내용 보존 + 압축 구조)

## 1. 설문조사서 개요

- 설문조사서: 응답자가 자신의 의견을 기입하는 조사 도구
- 목적: 통계조사 및 경영전략 수립을 위한 자료 수집
- 기능: 연구 결론 도출을 위한 핵심 수단
- 특징: 일정한 정보·지식·경험 보유 응답자로부터 데이터 수집

### 설문조사 예시

- 농림어업총조사: 5년 주기 전수조사
- 여론조사: 선거 지지율, 시청률
- 시장조사: 소비자 기호/만족도
- 품질조사: 불량률 조사
- 물가조사 등

### 효과

- 표준화된 설문 → 측정오차 감소
- 신속한 핵심 정보 확보
- 익명성 → 객관성 증가
- 표본 기반 가설검증 가능
- 시간/비용 절감

## 2. 설문조사 요건

- 객관적이고 적절한 측정방법 필요
- 응답자 특성 고려한 조사방법 선택
- 표본 대표성 확보 필수
- 분석 방법 사전 설계 필요
- 통계기법 적절히 적용
- 조사 목적에 맞는 정보만 수집
- 불필요 질문 제거
- 응답 가능성 고려

### 자료 종류

- 정량자료: 수치화 가능한 데이터
- 정성자료: 의견/태도
- 1차 자료: 직접 수집
- 2차 자료: 기존 데이터

## 3. 설문조사 방법

### 3.1 개별면접조사

- 장점: 심층 질문 가능, 추가 질문 가능, 정확도 높음, 비언어 정보 수집
- 단점: 비용 높음, 시간 소요, 면접원 훈련 필요

### 3.2 전화면접조사

- 장점: 비용 낮음, 빠름, 관리 용이, 응답률 상대적으로 높음
- 단점: 전화 미보유자 제외, 민감 질문 부적합, 시간 제한

### 3.3 집단조사

- 장점: 비용 절감, 빠른 수집, 관리 용이, 대인면접 효과
- 단점: 집단 영향, 심층질문 어려움, 장소 제약

### 3.4 CATI

- Computer Assisted Telephone Interviewing
- 컴퓨터 기반 전화 설문 시스템

### 면접 규칙

- 응답 경청
- 질문 숙지
- 편견 없는 태도
- 대화식 진행
- 관련 대화만 진행
- 응답 수준 맞춤
- 신뢰 형성

### 3.5 우편조사

- 장점: 비용 저렴, 접근성 높음, 심층 응답 가능
- 단점: 응답률 낮음, 주소 필요, 회수 어려움

### 3.6 전자설문

- 장점: 저비용, 빠른 회수, 자동 독촉 가능
- 단점: 인터넷 사용자 제한, 응답률 낮음, 개인정보 문제

## 4. 설문조사 과정

### 단계

- 연구주제/대상 선정
- 가설 설정
- 표본 선정
- 자료 수집 방법 결정

### 설문지 작성

- 변수 측정용 질문 구성
- 가설 기반 문항 설계
- 이해하기 쉬운 언어 사용
- 질문 순서 중요
- 개방/폐쇄형 선택

### 사전조사

- 본조사 전 테스트
- 문제점 확인 및 수정
- 문항/순서/표현 점검
- 대표성 있는 소규모 집단 대상
- 면접원이 문제 기록

### 자료분석

- 데이터 코딩
- 오류 검토
- 정량 분석 가능 형태 변환

## 5. 질문 형태

### 5.1 개방형 질문

- 자유 응답 방식
- 예: 미래 자동차 산업 전망
- 장점: 새로운 정보 확보, 심층 응답
- 단점: 분석 어려움, 코딩 필요, 무응답 가능성 높음

### 5.2 폐쇄형 질문

- 선택형 응답
- 예: ①~⑤ 선택
- 장점: 분석 용이, 응답 간편, 비교 가능
- 단점: 의견 제한, 편향 가능

### 5.3 부수적 질문

- 특정 조건 충족 시 추가 질문
- 예: 음주운전 경험 여부 → 추가 횟수 질문
- 특징: 맞춤형 정보 수집

## 6. 설문 작성 유의사항

### 기본 원칙

- 표준어 사용
- 객관적 문장 구성
- 간결한 문장 사용
- 한 문장 한 질문 원칙
- 중립적 표현 유지
- 타당성 검증 필수

### 금지 요소

- 긴 문장
- 모호한 표현
- 전문용어/속어/약어
- 편향 질문
- 특정 답 유도 질문

### 질문 순서

- 쉬운 질문 → 어려운 질문
- 일반 → 민감 질문
- 오리엔테이션 질문 먼저
- 인구통계 질문은 후반 배치
- 동일 척도는 그룹화

## 7. 응답 방식

- 폐쇄형: 가장 일반적, 분석 용이
- 개방형: 질적 정보 확보
- 리커트 척도: 동의 정도 측정
- 의미분별척도: 양극 형용사
- 순위척도: 중요도 순서
- 명목척도: 범주형 데이터

## 8. 설문 문항 배열 원칙

- 흥미 유발 질문 먼저
- 쉬운 질문 먼저 배치
- 핵심 질문은 중간
- 민감 질문은 후반
- 개인 의견 수집 포함
- 응답 피로도 고려

## 9. 핵심 결론

- 설문 품질 = 데이터 품질
- 질문 설계가 결과를 결정
- 표준화 + 객관성 + 중립성 필수
- 사전조사 필수
- 질문 구조가 응답률에 영향

# 🧠 시험 대비 요약본 (초압축)

- 설문조사 = 의견 → 데이터 수집 도구
- 목적: 통계 + 의사결정
- 방법: 면접 / 전화 / 집단 / 우편 / 전자
- 핵심 구조: 연구 → 가설 → 표본 → 설문 → 분석
- 질문 유형:
    - 개방형 = 심층
    - 폐쇄형 = 분석용
    - 부수질문 = 조건부
- 설문 핵심:
    - 표준어
    - 중립성
    - 간결성
    - 1문항 1개념
- 금지:
    - 유도 질문
    - 모호한 표현
    - 전문용어 남용
- 순서:
    - 쉬운 → 어려운
    - 일반 → 민감
- 핵심 원칙: 설문 품질 = 분석 결과 품질
