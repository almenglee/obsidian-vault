# 📚 1) 구조화된 정리본 (원문 최대 유지 + 시험용 구조 재배열)

## 1. 상관관계 개념

### 1-1. 상관관계의 정의와 의미

- 두 변수 간의 선형적 관련성을 측정하는 통계적 지표
- 변수들이 함께 어떻게 변화하는지를 나타냄
- 방향성과 강도를 동시에 표현
- 값 범위: -1 ~ 1

### 1-2. 상관계수 종류

- 피어슨 상관계수
    - 가장 널리 사용
    - 선형 관계만 측정
    - r = 1: 완전 양의 선형관계
    - r = -1: 완전 음의 선형관계
    - r = 0: 선형 관계 없음
- 켄달 상관계수
    - 순위 기반 상관계수
    - 비선형 관계에도 적용 가능
    - 순서 비교 기반 분석

### 1-3. 상관관계 vs 인과관계

- 상관관계: 변수 간 관련성만 의미
- 인과관계: 원인과 결과 관계
- 상관관계 ≠ 인과관계 (핵심 개념)

### 1-4. 상관관계 분석 한계

- 비선형 관계 포착 어려움
- 이상치(outlier)에 민감
- 제3변수(교란변수) 고려 부족

---

## 2. 회귀분석 기초

### 2-1. 회귀분석 정의

- 독립변수가 종속변수에 미치는 영향을 수학적으로 모델링하는 방법

### 2-2. 단순회귀모델

- 식: Y = β₀ + β₁X + ε
    - β₀: 절편
    - β₁: 기울기
    - ε: 오차항

### 2-3. 변수 정의

- 독립변수(X): 설명 변수, 원인
- 종속변수(Y): 결과 변수, 반응 변수

### 2-4. 회귀계수 해석

- β₁: X가 1 증가할 때 Y의 평균 변화량

### 2-5. 결정계수 (R²)

- 모델 설명력 (0~1)
- 높을수록 데이터 설명력 우수

---

## 3. 다중회귀분석

### 3-1. 개념

- 여러 독립변수가 종속변수에 미치는 영향 분석
- 복잡한 현상 설명 가능

### 3-2. 모델

- Y = β₀ + β₁X₁ + β₂X₂ + ... + βₚXₚ + ε
- 각 β는 다른 변수 통제 상태에서의 효과

### 3-3. 변수 선택 방법

- 전진 선택법
- 후진 제거법
- 단계적 선택법
- 모든 가능한 회귀

### 3-4. 다중공선성 문제

- 독립변수 간 높은 상관관계
- 회귀계수 불안정
- 진단: VIF
- 해결: 변수 제거 / 변환

---

## 4. 회귀분석 해석

### 4-1. 회귀계수 해석

- 다른 변수 고정 시 특정 변수 효과
- 표준화 계수로 비교 가능

### 4-2. t-검정

- t = 회귀계수 / 표준오차
- 개별 변수 유의성 검정

### 4-3. p-value

- p < 0.05 → 통계적으로 유의

### 4-4. F검정 (ANOVA)

- 전체 모델 유의성 검정

### 4-5. 설명력

- R²: 설명력
- Adjusted R²: 변수 수 보정
- 0.7 이상이면 높은 설명력

---

## 5. 상관 vs 회귀 비교

### 5-1. 상관분석

- 두 변수 관계 강도 측정
- 인과관계 아님
- 대칭적 관계 (X↔Y 동일)

### 5-2. 회귀분석

- 독립 → 종속 방향성 존재
- 인과 추론 가능(제한적)
- 비대칭 관계

---

## 6. 활용 분야

### 6-1. 마케팅

- 광고비 vs 매출 분석
- 가격 탄력성
- 소비자 행동 분석

### 6-2. 교육/심리

- 성취도 예측
- 심리 변수 관계 분석
- 교육 효과 평가

### 6-3. 의학/생물학

- 질병 위험 요인 분석
- 약물 효과 예측
- 유전자 분석

---

## 7. 분석 절차 (엑셀 기준)

- 데이터 정리 및 결측치 처리
- 상관분석 실행
- 회귀분석 실행
- p-value / R² 확인
- 산점도 및 추세선 시각화

---

## 8. 결론

- 상관관계: 관계 “측정”
- 회귀분석: 관계 “모델링 및 예측”
- 인과관계 해석은 신중해야 함
- 잔차 분석 및 가정 검토 필수

# 🧠 2) 시험 대비 요약본 (핵심 압축 + 암기용)

## 📌 상관관계 핵심

- -1 ~ 1 범위
- 피어슨: 선형 관계
- 켄달: 순위 기반
- 핵심: “상관 ≠ 인과”

## 📌 회귀분석 핵심

- Y = β₀ + β₁X + ε
- β₁ = X 1 증가 시 Y 변화량
- R² = 설명력 (0~1)

## 📌 다중회귀 핵심

- 변수 여러 개 사용
- β는 “다른 변수 통제된 효과”
- 다중공선성 = 변수 간 높은 상관 → 문제 발생

## 📌 통계 검정 핵심

- t-test: 개별 변수
- F-test: 전체 모델
- p < 0.05 → 유의

## 📌 반드시 구분

- 상관분석: 관계 “있나?”
- 회귀분석: “얼마나 영향?”

## 📌 시험 단골 포인트

- 상관관계 ≠ 인과관계 (최중요)
- R² 해석
- p-value 의미
- 다중공선성(VIF)
- 회귀계수 해석
