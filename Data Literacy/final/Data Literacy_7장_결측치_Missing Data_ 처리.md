# 📚 교본용 정리본 (내용 보존 + 압축 구조)

## 1. 결측치 개념

- 결측치(Missing Data): 값이 비어 있는 상태(Blank)
- 데이터 분석에서 가장 기본적으로 처리해야 하는 문제
- 발생 원인: 데이터 누락, 응답 생략, 시스템 오류
- 엑셀에서 빈 셀 형태로 나타남
- 결측치는 데이터 분석 정확성을 저해하는 주요 요인

## 2. 결측치 문제점

### 2.1 계산 오류

- 평균/합계 계산 오류 발생
- 누락값으로 인한 분모/분자 불일치
- 통계 함수 결과 왜곡
- #VALUE! 오류 발생 가능

### 2.2 기능 오작동

- 정렬 시 빈 셀 위치 문제
- 필터 결과 누락
- 피벗 테이블 오류
- 데이터 범위 인식 오류

### 2.3 분석 결과 왜곡

- 표본 크기 감소
- 편향된 결과 발생
- 상관관계 분석 왜곡
- 패턴 인식 어려움

### 2.4 시각화 오류

- 차트 공백 발생
- 그래프 선 끊김
- 추세선 왜곡
- 시각적 해석 오류

### 2.5 의사결정 영향

- 데이터 품질 저하
- 잘못된 의사결정
- 비즈니스 손실
- 보고서 신뢰도 하락

## 3. 결측치 찾는 방법

### 3.1 조건부 서식

- 데이터 범위 선택
- 홈 → 조건부 서식 → 새 규칙
- 빈 셀 선택
- 색상 지정으로 시각화

### 3.2 COUNTBLANK 함수

- 구문: =COUNTBLANK(범위)
- 빈 셀 개수 계산
- 예: =COUNTBLANK(A1:G101)
- 용도: 결측치 수량 파악, 데이터 품질 점검

## 4. COUNTBLANK 특징

- 범위 내 빈 셀 개수 계산
- 결측치 현황 빠르게 파악 가능
- 대용량 데이터 분석에 유용
- 결측치 비율 계산 가능
- 데이터 정리 전 상태 확인 가능

## 5. 결측치 처리 (초급)

### 5.1 삭제

- 결측치 많은 행 삭제
- 데이터 품질 향상
- 분석 정확도 증가
- 주의: 중요 데이터 삭제 금지
- 삭제 전 백업 필요
- 10% 이상 삭제 시 재검토

### 5.2 수동 대체

- 빈 셀 직접 입력
- 데이터 특성 기반 값 대체
- 소규모 데이터 적합
- 직관적 방법

### 숫자 데이터 대체

- 평균값 대체
- 중앙값 대체
- 최빈값 대체
- 0 대체
- 최소/최대값 활용
- 인접값 참조

### 문자 데이터 대체

- “미기입”, “알 수 없음”, “N/A”
- 기본값 설정
- 공통 텍스트 입력
- 빈칸 유지 경우도 존재

## 6. 함수 활용 대체

### 문자 데이터

- =IF(A2="", "No Email", A2)
- 빈 값 → 특정 문자열 대체

### 평균 대체

- =AVERAGE(B2:B500)
- 전체 평균으로 채움

### 중앙값 대체

- =MEDIAN(B2:B500)
- 이상치 영향 감소

## 7. 실습 과정

### 7.1 결측치 찾기

- 데이터 범위 선택
- 필터 → Blanks 선택
- 결측치 행만 표시
- 조건부 서식 활용
- COUNTBLANK 활용

### 7.2 결측치 처리

- 숫자: 평균값 대체
- 문자: 기본 텍스트 입력
- 날짜: 인접값 참조
- 중요 데이터 개별 검토

### 7.3 결과 확인

- 처리 전후 비교
- 통계 요약 비교
- 시각화 확인
- 품질 평가

### 7.4 추가 연습

- 다른 데이터셋 적용
- 다양한 대체 방법 테스트
- 이상치 추가 점검

# 🧠 시험 대비 요약본 (초압축)

- 결측치 = 빈 데이터
- 문제: 계산 오류 + 분석 왜곡 + 의사결정 오류
- 핵심 영향: 평균/합계/상관관계 모두 왜곡
- 찾기:
    - 조건부 서식
    - COUNTBLANK
- 삭제: 간단하지만 정보 손실 위험
- 대체:
    - 평균/중앙값/최빈값
    - 문자: N/A, 미기입
- 함수:
    - IF + 빈값 처리
    - AVERAGE / MEDIAN
- 핵심 원칙: 결측치 처리는 “삭제보다 보정이 우선”
