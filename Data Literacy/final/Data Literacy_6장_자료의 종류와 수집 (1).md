# 📚 구조화된 정리본 (내용 보존 + 압축형)

## 1. 데이터 분석의 중요성

- 데이터 분석의 중요성: 데이터 기반 의사결정 시대
- 엑셀 시대의 종말: 단순 데이터 처리에서 AI 활용 능력 요구로 변화
- 현대 비즈니스: 데이터 분석 + AI 활용 필수
- 데이터 분석 정의: 수집된 데이터에서 의미 있는 패턴과 인사이트 발견 과정
- 통계학 역할: 데이터를 정보로 전환하는 방법론
- 흐름: 데이터 → 통계학 → 정보 → 가치 창출

## 2. 자료의 종류 (전체 구조)

- 변수 수 기준: 단변수 자료 / 다변수 자료
- 측정 방식 기준: 질적 자료 / 양적 자료
- 시간 기준: 시계열 자료 / 횡단면 자료

## 3. 단변수 자료 vs 다변수 자료

- 단변수 자료: 특정 변수가 하나인 자료
- 분석 가능 요소: 대표값, 기본단위 동질성, 특이관측 존재 여부
- 예: 특정 지역 거주자의 소득 자료
- 다변수 자료: 변수 둘 이상 존재
- 분석 요소: 변수 간 관련성, 관련성 정도, 미래값 예측
- 예: 광고인지 여부 vs 상품구입 여부
- 사례:
    - 백화점 100명 광고인지/구매 여부 → 광고효과 분석
    - 벤처기업 성장률/사업분야/기술력 → 성공요인 분석
    - 종업원 급여/성별/근속/직무/교육 → 성별 임금차 분석

## 4. 자료 분류 (질적/양적/시간)

- 자료 분류 기준:
    - 변수 수: 단/다변수
    - 측정 방식: 질적/양적
    - 측정 시점: 시계열/횡단면

## 5. 질적 자료 (정성적 자료)

- 범주형 자료: 수치 의미 없음
- 숫자 부여해도 크기 의미 없음
- 분석 시 빈도 중심 접근 필요

### 5.1 명목자료

- 정의: 단순 범주 분류 자료
- 특징: 순서 없음, 크기 없음
- 예: 성별, 결혼상태, 직업
- 분석: 빈도, 비율

### 5.2 서수자료

- 정의: 순서/순위 존재하는 자료
- 특징: 순위 의미 있으나 산술연산 불가
- 예: 학점, 만족도, 기업 크기(대/중/소), 경쟁력 순위
- 분석: 순위비교, 중앙값, 비모수 검정

## 6. 양적 자료 (정량적 자료)

- 정의: 수치로 표현 가능한 자료
- 특징: 평균, 분산 등 수치 연산 가능
- 예: 체중, 거리, 시간, 불량품 개수

### 6.1 이산자료

- 정의: 발생 횟수/정수값 자료
- 특징: 양의 정수값
- 예: 자녀 수, 교통사고 건수

### 6.2 연속자료

- 정의: 무한히 작은 단위 측정 가능
- 특징: 실수값 가능
- 예: 투자수익률, 부채비율, 불량률, 시장점유율

## 7. 시계열 / 횡단면 / 패널 자료

- 시계열 자료: 시간 순서 중요, 시간 변화 분석
- 예: 월 생산실적, 분기 판매실적, 주식가격
- 특징: 추세, 계절성, 주기성 분석
- 횡단면 자료: 동일 시점 여러 대상 비교
- 예: 특정 시점 기업 실적, 작업반 생산성
- 패널 자료: 시계열 + 횡단면 결합
- 예: 2005~2009 국가별 휴대전화 가입자 수
- 패널 활용: 성장률 비교, 시간 변화, 국가 간 격차 분석

## 8. 자료 수집 방법

- 관측(Observation): 직접 관측, 간단하지만 통제 어려움
- 실험(Experiment): 변수 통제, 인과관계 분석 가능, 비용/시간 큼
- 조사(Survey): 전수조사/표본조사, 설문/전화/인터넷 등

## 9. 표본추출 방법

### 9.1 확률표본추출

- 정의: 모든 개체가 동일 확률로 선택
- 방법: 단순임의, 계통, 층화, 집락

#### 단순임의추출

- 난수로 표본 선택
- 특징: 모든 개체 동등 확률
- 장점: 대표성 높음, 통계적 추론 적합
- 단점: 시간/비용 큼, 대규모 적용 어려움

#### 계통추출

- 일정 간격으로 표본 선택
- 첫 개체는 난수 선택 후 일정 간격 적용
- 장점: 단순임의보다 효율적

#### 집락추출

- 모집단을 집락으로 나누고 일부 집락 전체 조사
- 특징: 집락 내 이질적, 집락 간 동질적
- 장점: 비용 절감, 현장 조사 용이
- 단점: 대표성 왜곡 가능, 표본오차 큼

#### 층화추출

- 모집단을 계층으로 나누고 각 층에서 추출
- 특징: 층 내 동질적, 층 간 이질적
- 절차: 계층 구분 → 각 층에서 단순임의추출
- 장점: 대표성 높음, 표본오차 감소
- 단점: 계층 구분 필요

### 9.2 비확률표본추출

- 정의: 조사자 판단 기반 추출
- 방법: 임의추출, 판단추출, 할당추출, 누적추출
- 특징: 경제성/시간 절약, 대표성 낮을 수 있음

## 10. 표본추출오차

- 정의: 표본이 모집단과 다르기 때문에 발생하는 오차
- 원인: 표본 크기 부족, 추출 방법 문제
- 특징: 다른 표본 선택 시 결과 변화
- 해결: 표본 크기 증가 (단, 비용 증가)

## 11. 비표본추출오차

- 정의: 표본 설계 외 과정에서 발생하는 오차

### 유형

- 자료 취득 오류: 측정/기록 실수, 질문 오해
- 응답편의 오차: 설문 문구/조사원 영향
- 무응답편의 오차: 응답 거부/불가능
- 선택편의 오차: 특정 집단 배제

### 관리 방법

- 조사원 교육 강화
- 설문지 사전 검증
- 다양한 접촉 방법
- 무응답 보정
- 파일럿 테스트

## 12. 통계의 함정

- 표본 대표성 문제
- 비교 기준 동일성 문제
- 상관관계 ≠ 인과관계

## 13. 요약

- 자료 종류: 단/다변수, 질적/양적, 시계열/횡단면
- 수집 방법: 관측/실험/조사
- 표본추출: 확률/비확률
- 오차: 표본오차 + 비표본오차
- 핵심: 표본 설계가 결과를 결정

# 🧠 시험 대비 요약본 (초압축)

- 데이터 분석 = 패턴 발견 + 의사결정
- 통계 = 데이터 → 정보 변환
- 자료 축: 변수 / 측정 / 시간
- 단변수 vs 다변수 = 관계 분석 여부
- 질적: 명목/서수
- 양적: 이산/연속
- 시계열 = 시간 흐름
- 횡단면 = 동일 시점 비교
- 패널 = 시간+대상 결합
- 표본추출: 단순/계통/층화/집락
- 비확률: 판단/임의 등
- 표본오차 = 샘플링 문제
- 비표본오차 = 조사 과정 문제
- 핵심 위험: 대표성 붕괴, 선택편의
- 핵심 원칙: 설계 > 분석
