# 1. 시험 핵심 구조 (이 과목의 본질)

이 강의는 구조가 매우 명확합니다.

> **그래프 기반 분석 + 통계량 기반 분석 (기술통계)**

즉 시험은 반드시 아래 두 축으로 나옵니다:

### ① 그래프/표 해석

- 도수분포표
    
- 히스토그램 / 막대 / 원그래프
    
- 왜곡된 그래프 판별
    

### ② 수치 계산 (핵심)

- 평균 / 중앙값 / 최빈값
    
- 분산 / 표준편차
    
- 사분위수 / 백분위수
    

👉 결론:  
**“계산 + 해석” 둘 다 요구됨 (단순 암기 절대 부족)**

# 2. 필수 개념 (시험 90% 커버)

## 2.1 중심경향 (무조건 나옴)

- 평균 (Mean)
    
- 중앙값 (Median)
    
- 최빈값 (Mode)
    

핵심 포인트:

- 평균: 이상치(outlier)에 매우 민감
    
- 중앙값: 왜곡 데이터에서 대표값
    
- 최빈값: 범주형 데이터에서 사용
    

👉 시험 포인트

- “어떤 상황에서 무엇을 써야 하는가”
    
- 단순 계산보다 **선택 문제** 많이 나옴
    

---

## 2.2 산포 (시험 핵심)

- 범위 (Range)
    
- 분산 (Variance)
    
- 표준편차 (Standard Deviation)
    

핵심 개념:

- 범위 = max - min (극단값에 민감)
    
- 분산 = 편차² 평균
    
- 표준편차 = 분산의 루트
    

👉 시험 포인트

- 계산 문제 거의 확정
    
- “두 데이터 비교” 문제 자주 나옴
    

---

## 2.3 상대적 위치

- 사분위수 (Q1, Q2, Q3)
    
- 백분위수
    
- 사분위 범위 (IQR = Q3 - Q1)
    

👉 시험 포인트

- “중앙 50% 범위” = IQR
    
- box plot 해석 가능성 있음
    

---

## 2.4 분포 형태

- 왜도 (Skewness)
    
- 첨도 (Kurtosis)
    

👉 시험 포인트

- 오른쪽 꼬리 → 양의 왜도
    
- 왼쪽 꼬리 → 음의 왜도
    

---

## 2.5 도수분포표 (실습 핵심)

- 계급 (class)
    
- 도수 (frequency)
    
- 상대도수 = 도수 / 전체
    
- 누적상대도수
    

👉 시험 포인트

- 표 완성
    
- 그래프 연결
    

---

## 2.6 Excel 핵심 함수 (실습 문제)

반드시 암기:

```
=AVERAGE()
=MEDIAN()
=MODE()
=VAR(), =VARP()
=STDEV(), =STDEVP()
=COUNTIF()
```

👉 시험 포인트

- 함수 자체보다:  
    → “어떤 상황에서 어떤 함수 선택하는가”

# 3. 실습 문제 유형 분석 (출제 패턴)

## 유형 1: 도수분포표 만들기

- 구간 나누기
    
- COUNTIF 사용
    

👉 난이도: 중

---

## 유형 2: 상대도수 / 누적

```
상대도수 = 도수 / 전체
```

👉 함정:

- 절대참조 ($) 사용 여부
    

---

## 유형 3: 평균/표준편차 계산

- 직접 계산 or Excel
    

👉 거의 확정 출제

---

## 유형 4: 그래프 해석

- 히스토그램 보고 분포 판단
    
- 왜곡된 그래프 찾기
    

👉 개념형 문제

---

## 유형 5: 통계량 선택 문제 (중요)

예:

- 소득 데이터 → 중앙값
    
- 성장률 → 기하평균
    

👉 교수들이 매우 좋아하는 문제

---

## 유형 6: 비교 문제

- 두 데이터 중 변동성 큰 것
    
- 평균 같고 분산 다른 케이스
    

👉 사고력 문제

# 4. 시험에서 가장 중요한 사고 패턴

## 핵심 프레임 (이걸 기억하십시오)

### 1단계: 데이터 유형 파악

- 범주형 vs 수치형
    

### 2단계: 목적 파악

- 대표값? → 평균/중앙값
    
- 퍼짐? → 표준편차
    
- 위치? → 사분위
    

### 3단계: 도구 선택

- Excel 함수 or 수식

# 5. 시험 전략 (실전용)

## 전략 1: 계산 문제 = 무조건 맞춘다

- 평균 / 분산 / 표준편차 → 실수 없이
    

👉 여기서 점수 갈립니다

---

## 전략 2: “선택 문제” 대비

|상황|정답|
|---|---|
|이상치 있음|중앙값|
|성장률|기하평균|
|데이터 퍼짐 비교|표준편차|

---

👉 이거 틀리면 실습 문제 전부 날립니다
