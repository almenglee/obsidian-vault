# === 제01장_물리학과측정 ===

## 1. 물리학 개요

- 자연현상의 **기본 법칙 탐구 + 수학적 모델링**
    
- 핵심: **이론 ↔ 실험 ↔ 수학**
    

---

## 2. SI 기본 단위

|물리량|단위|
|---|---|
|길이|m|
|질량|kg|
|시간|s|
|전류|A|
|온도|K|
|물질량|mol|
|광도|cd|

### 현대 정의 (중요)

- 시간: 세슘 원자 진동
    
- 길이: 빛의 속도 기반
    
- 질량: 플랑크 상수 기반 (재정의됨)
    

---

## 3. 유도량

- 속력:  
    [  
    v = \frac{L}{T}  
    ]
    
- 밀도:  
    [  
    \rho = \frac{m}{V}  
    ]
    

---

## 4. 모형화

- 입자 모델: 크기 무시
    
- 핵심: **불필요한 요소 제거**
    

---

## 5. 차원 분석

- 기본 차원:
    
    - 길이 [L], 질량 [M], 시간 [T]
        

### 규칙

- 같은 차원만 더함
    
- 식 좌우 차원 동일
    

예:  
[  
v = at \Rightarrow [L/T] = [L/T^2] \cdot [T]  
]

---

## 6. 단위 환산

핵심: **변환계수 = 1**

예:  
[  
15.0,in \times \frac{2.54,cm}{1,in}  
]

---

## 7. 크기 추정 (Order of Magnitude)

- 과학적 표기:  
    [  
    a \times 10^n  
    ]
    
- 기준:
    
    - ( a < 3.16 \Rightarrow 10^n )
        
    - ( a > 3.16 \Rightarrow 10^{n+1} )
        

---

## 8. 유효숫자

### 규칙

- 0 제외 숫자 → 항상 유효
    
- 사이 0 → 유효
    
- 소수 끝 0 → 유효
    

### 계산 규칙

- 덧셈: **소수 자리 기준**
    
- 곱셈: **유효숫자 개수 기준**
    

---

# === 제02장_일차원운동 ===

## 1. 위치 / 변위 / 거리

- 위치: ( x(t) )
    
- 변위:  
    [  
    \Delta x = x_f - x_i  
    ]
    
- 거리: 스칼라
    

---

## 2. 속도

### 평균 속도

[  
v_{avg} = \frac{\Delta x}{\Delta t}  
]

### 순간 속도

[  
v = \frac{dx}{dt}  
]

---

## 3. 가속도

### 평균

[  
a_{avg} = \frac{\Delta v}{\Delta t}  
]

### 순간

[  
a = \frac{dv}{dt}  
]

---

## 4. 등속 운동

[  
x = x_0 + vt  
]

---

## 5. 등가속도 운동 (핵심 3식)

[  
v = v_0 + at  
]

[  
x = x_0 + v_0 t + \frac{1}{2}at^2  
]

[  
v^2 = v_0^2 + 2a(x - x_0)  
]

---

## 6. 자유 낙하

[  
a = -g \quad (g = 9.8,m/s^2)  
]

---

- 스칼라: 크기만
    
- 벡터: 크기 + 방향
    

---

]

]
