# #1. 꿀벌 농장의 꿀 생산량

꿀벌 농장에서는 N마리의 꿀벌이 동시에 꿀을 생산하고 있습니다. 1번 꿀벌은 하루에 1그램, 2번 꿀벌은 하루에 2그램의 꿀을 생산합니다. 이와 같이, 꿀벌의 번호에 해당하는 그램 수만큼 매일 꿀을 생산합니다. D일 동안 모든 꿀벌이 생산한 꿀의 총량(그램)을 계산하세요.

## 입력 예시

첫 번째 줄에 꿀벌의 수 N이 주어집니다.

두 번째 줄에 작업한 날 수 D가 주어집니다.

## 출력 예시

N마리의 꿀벌이 D일 동안 생산한 총 꿀의 양을 출력하세요.

|입력 예시|출력 예시|
|---|---|
|5  <br>3|45|
|4  <br>2|20|
|3  <br>4|24|

## Answer

```python
bees, days = int(input()), int(input())

print(days*bees*(bees+1)//2)
```

# #2. 텃밭 확장 계획

농부는 직사각형 텃밭을 운영하고 있습니다. 텃밭의 초기 가로 길이는 W미터, 세로 길이는 H미터입니다. 농부는 매일 가로 길이만 A미터씩 확장합니다. D일 후 텃밭의 면적을 계산하는 프로그램을 작성하세요.

## 입력 예시

첫 번째 줄에 텃밭의 초기 가로 길이 W가 주어집니다.

두 번째 줄에 텃밭의 세로 길이 H가 주어집니다.

세 번째 줄에 매일 확장되는 가로 길이 A가 주어집니다.

네 번째 줄에 확장 작업을 진행한 날 수 D가 주어집니다.

## 출력 예시

D일 후 텃밭의 면적을 출력하세요.

|입력 예시|출력 예시|
|---|---|
|10  <br>5  <br>2  <br>3|80|
|8  <br>3  <br>1  <br>5|39|
|15  <br>4  <br>2  <br>2|76|

## Answer

```python
w,h,a,d =int(input()), int(input()), int(input()), int(input())

print(h*(w+a*d))
```

# #3. 체육대회 달리기 기록 분석title

N명의 학생이 체육대회 달리기에 참가했습니다. 각 학생의 기록(초 단위)을 입력받아, 전체 학생의 평균 기록과 가장 빠른(가장 작은) 기록을 계산하는 프로그램을 작성하세요.

## 입력 예시

첫 번째 줄에 학생의 수 N이 주어집니다.

두 번째 줄에 각 학생의 기록이 공백으로 구분되어 주어집니다. 기록은 양의 정수입니다.

## 출력 예시

첫 번째 줄에 평균 기록을 소수점 둘째 자리까지 출력하세요.

두 번째 줄에 가장 빠른 기록을 출력하세요.

|입력 예시|출력 예시|
|---|---|
|4  <br>12 15 11 14|Average: 13.00  <br>Fastest: 11|
|3  <br>25 30 20|Average: 25.00  <br>Fastest: 20|
|5  <br>8 9 7 10 6|Average: 8.00  <br>Fastest: 6|
|6  <br>100 95 88 92 97 90|Average: 93.67  <br>Fastest: 88|
|2  <br>45 50|Average: 47.50  <br>Fastest: 45|

## Answer

```python
num = int(input())

  

arr = [int(x) for x in input().strip().split(" ")]

print(f"Average: {sum(arr)/num:.2f}")

print("Fastest:",min(arr))
```

# #4. 택배 분류 시스템

물류 센터에서 택배 상자를 분류해야 합니다. 각 상자에는 고유 번호가 있으며, 3의 배수인 번호의 상자를 먼저, 나머지 상자를 나중에 배치해야 합니다. 주어진 상자 번호를 입력받아 분류하는 프로그램을 작성하세요.

## 입력 예시

첫 번째 줄에 상자의 개수 N이 주어집니다.

두 번째 줄에 N개의 상자 번호가 공백으로 구분되어 주어집니다. 번호는 1 이상 100 이하의 정수입니다.

## 출력 예시

3의 배수 번호는 앞에, 나머지 번호는 뒤에 배치된 상자 번호를 공백으로 구분하여 출력하세요.

|입력 예시|출력 예시|
|---|---|
|6  <br>5 9 2 6 7 3|9 6 3 5 2 7|
|5  <br>1 3 6 8 12|3 6 12 1 8|
|4  <br>10 11 12 13|12 10 11 13|

## Answer

```python
num = int(input())

arr = [int(x) for x in input().strip().split(" ")]

  

mul3 = []

res = []

  

for x in arr:

    if x%3==0:

        mul3.append(x)

    else:

        res.append(x)

  

print(*(mul3+res))
```

# #5. 비밀 편지 해독

스파이가 남긴 비밀 편지를 해독하려면, 먼저 문자열에서 모든 모음(a, e, i, o, u, A, E, I, O, U)을 '*'로 바꾼 뒤, 문자열 전체를 뒤집어야 합니다. 이 편지를 해독하는 프로그램을 작성하세요.

## 입력 예시

첫 번째 줄에 비밀 편지 문자열이 주어집니다. 문자열의 길이는 100 이하이며, 공백 없이 알파벳으로만 구성됩니다.

## 출력 예시

모음을 '*'로 바꾼 뒤 뒤집힌 문자열을 출력하세요.

|입력 예시|출력 예시|
|---|---|
|Hello|*ll*H|
|Apple|*lpp*|
|wOrLD|DLr*w|
|Python|n*htyP|
|ScIeNcE|*cN**cS|

## Answer

```python
print("".join("*" if ch.lower() in "aeiou" else ch for ch in input()[::-1]))
```

# #6. 보물 상자 속 완전제곱수

탐험가는 보물 상자에서 N개의 숫자를 발견했습니다. 이 중에서 완전제곱수만을 찾아야 보물 상자를 열 수 있습니다. 완전제곱수(Perfect Square)란 어떤 정수의 제곱으로 표현할 수 있는 수입니다 (예: 1=1², 4=2², 9=3², 16=4², 25=5², …). 주어진 숫자들 중 완전제곱수만을 출력하는 프로그램을 작성하세요.

## 입력 예시

첫 번째 줄에 숫자의 개수 N이 주어집니다.

두 번째 줄에 N개의 숫자가 공백으로 구분되어 주어집니다. 숫자는 1 이상 10000 이하입니다.

## 출력 예시

완전제곱수인 숫자들만을 입력 순서대로 출력하세요. 완전제곱수가 없을 경우 "None"을 출력합니다.

> 힌트: 어떤 수 X가 완전제곱수인지 확인하려면, 1부터 차례로 제곱해 보면서 X와 같아지는지 비교하면 됩니다.

| 입력 예시                        | 출력 예시     |
| ---------------------------- | --------- |
| 8  <br>4 7 10 16 15 25 23 24 | 4 16 25   |
| 6  <br>5 9 11 36 17 49       | 9 36 49   |
| 4  <br>3 6 8 10              | None      |
| 7  <br>1 2 3 4 5 6 7         | 1 4       |
| 5  <br>81 64 50 100 99       | 81 64 100 |

## Answer

```python
num = int(input())

  

def is_square(x):

    for i in range(1,100,1):

        if i**2 ==x:

            return True

  

    return False

  

arr = [int(x) for x in input().strip().split()]

  

sqrs = [str(x) for x in arr if is_square(x)]

print("None" if len(sqrs) == 0 else " ".join(sqrs))
```
